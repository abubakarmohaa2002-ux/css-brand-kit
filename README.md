# Brand Kit

A simple HTML and CSS project that demonstrates how to build a consistent typography system using Google Fonts. This project focuses on creating a clean landing page by applying font families, font sizes, font weights, spacing, and button styling to establish a professional brand identity.

## Features

- Google Fonts integration
- Consistent typography system
- Display font for headings
- Readable body font
- Styled brand label
- Call-to-action button
- Responsive page layout
- Card-based content section

## Technologies Used

- HTML5
- CSS3
- Google Fonts

```text
brand-kit/
├── index.html

└── README.md
```

## Concepts Practiced

### Google Fonts
Imported custom fonts from Google Fonts to improve the visual appearance and consistency of the webpage.

### Typography System
Applied separate font families for headings and body text to create a clear visual hierarchy.

### Font Styling
Used font size, font weight, line height, and letter spacing to improve readability and design.

### Text Transform
Applied uppercase styling to create a recognizable brand identity.

### Button Styling
Created a call-to-action button using typography, spacing, colors, and text decoration.

### Responsive Layout
Organized content with proper spacing and flexible layouts for a clean user experience.
